# 📖 [Recipe-app] <a name="about-project"></a>

**[Recipe-app]** is a recipe web application. User can create accounts, recipe and foods can see public_recipes and can generate shopping list. Built with ruby on rails.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```

### Setup

Clone this repository to your desired folder:

```sh
  git clone git@github.com:Htetaungkyaw71/Recipe-app.git

```

### Install

Install this project with:

```sh
  cd Recipe-app
  bundle install
  rails db:create
  rails db:migrate
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rspec spec
```
